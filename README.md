# Counter DApp

A simple Ethereum DApp that increments and decrements a counter.

## Tech Stack
- Solidity
- Ethereum (Sepolia Testnet)
- Remix IDE
- MetaMask
- Web3.js
- HTML & CSS

## How it works
- Users connect their wallet using MetaMask
- Clicking + or - sends a transaction
- Transactions are stored on the Ethereum blockchain
